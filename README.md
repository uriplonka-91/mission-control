# Mission Control

Your personal command center dashboard built with Next.js.

## Features

- **Dashboard** — Overview of tasks and stats
- **Task Manager** — Create, complete, and delete tasks
- **Real-time Updates** — All data persisted to localStorage
- **Clean UI** — Dark theme with Tailwind CSS

## Getting Started

### Prerequisites
- Node.js 14+
- npm or yarn

### Installation

```bash
cd C:\Users\URI-AI\mission-control
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
